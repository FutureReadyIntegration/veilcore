## What Is VeilCore?

VeilCore is a living security platform designed specifically for healthcare infrastructure. Unlike traditional security tools that react to threats, VeilCore uses an organ-based architecture where 82 specialized security components work together continuously to detect, analyze, and eliminate vulnerabilities before they become breaches.

**Built solo by Marlon Ástin Williams in 2025-2026.**

**Hospitals are under attack:**

## The Organ Architecture

VeilCore uses a biological metaphor - just like your body has organs that perform specialized functions, VeilCore has 82 security organs working together:

### P0 - Critical Organs (8)
The core security functions that must never fail:
- **Guardian** - Authentication gateway
- **Sentinel** - Threat detection
- **Audit** - Compliance monitoring
- **Chronicle** - Blockchain audit trail
- **Cortex** - Central decision engine
- **Insider Threat** - Detects internal attacks
- **PHI Classifier** - Identifies protected health information
- **Encryption Enforcer** - Ensures data protection

### P1 - High Priority Organs (10)
Essential security controls:
- **Watchdog** - System health monitoring
- **Firewall** - Network protection
- **Backup** - Data resilience
- **Quarantine** - Threat isolation
- **Vault** - Secrets management
- **MFA** - Multi-factor authentication
- **RBAC** - Role-based access control
- **Host Sensor** - Endpoint security
- **Network Monitor** - Traffic analysis
- **Threat Intel** - Intelligence gathering

### P2 - Specialized Organs (64)
Healthcare, network, monitoring, scanning, system, and recovery organs working 24/7.

## Quick Start

**Prerequisites:**
- Ubuntu 24.04 LTS
- Python 3.12+
- Systemd
- 4GB RAM minimum

**Installation:**
```bash
git clone https://github.com/futurereadyintegration/veilcore.git
cd veilcore
./install.sh
sudo systemctl start veil-dashboard
```

**Access dashboard:** http://localhost:8000

## Key Features

### Real-Time Threat Detection
VeilCore continuously scans your infrastructure and eliminates threats as they appear:
- Suspicious processes automatically quarantined
- Unauthorized file access blocked
- Privilege escalation attempts detected
- Data exfiltration stopped
- USB devices monitored
- Security tampering prevented

### Righteousness Engine
Context-aware ethical decision making. Should a nurse access a patient file at 3 AM? VeilCore doesn't just check permissions - it evaluates time of access, user role, patient relationship, emergency context, and historical patterns.

### Blockchain Audit Trail
Every security decision logged in an immutable ledger. Cannot be tampered with. Perfect for HIPAA audits.

### Healthcare Integration
Built specifically for hospital infrastructure:
- **EPIC** - EMR system integration
- **Imprivata** - Single sign-on bridge
- **HL7** - Healthcare messaging filter
- **FHIR** - API gateway protection
- **DICOM** - Medical imaging shield
- **IoMT** - Medical device protection
