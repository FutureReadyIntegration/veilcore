# Security Policy

## Reporting a Vulnerability

If you discover a security issue in VeilCore, please report it privately to the project owner Future_ReadyIntegration@outlook.com before public disclosure.

## Scope

This policy applies to:

- VeilCore core platform
- organ / engine architecture
- API
- dashboard
- terminal
- event pipeline

## Disclosure

Please avoid public disclosure until the issue can be reviewed and addressed.
