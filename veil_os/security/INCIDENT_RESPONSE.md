
# Incident Response — Veil OS Organ Engine

## Triggers
- Unexpected configuration changes
- Audit log anomalies
- Operator error

## Response
1. Preserve audit logs
2. Identify command, user, target
3. Review documentation version
4. Remediate and document

## Postmortem
- Documentation reviewed
- Procedures updated if needed

Supports SOC 2 and ISO incident handling.
