# VeilCore - Hospital Security Platform

**Built by Marlon Ástin Williams**

VeilCore is a living security platform designed specifically for healthcare infrastructure. It protects hospitals from cyber threats, insider attacks, and compliance violations through 82 active security organs working continuously.

## Features

- **82 Active Security Organs** - Continuous scanning and threat detection
- **Real-Time Threat Detection** - Identify and eliminate vulnerabilities as they occur
- **Insider Threat Detection** - Catch malicious activity from trusted users
- **Righteousness Engine** - Ethical decision-making framework for access control
- **Blockchain Audit Trail** - Immutable logging for HIPAA compliance
- **Healthcare Integration** - Built for EPIC, Imprivata, HL7, FHIR, DICOM

## Architecture

VeilCore uses an organ-based architecture where specialized security components work together:

- **P0 Organs (Critical):** Guardian, Sentinel, Audit, Chronicle, Cortex, Insider Threat, PHI Classifier, Encryption Enforcer
- **P1 Organs (High Priority):** Watchdog, Firewall, Backup, Quarantine, Vault, MFA, RBAC, Host Sensor, Network Monitor, Threat Intel
- **P2 Organs (Healthcare & Operations):** 63+ specialized organs for healthcare security, network protection, monitoring, and recovery

## Technology Stack

- Python 3.12
- FastAPI
- Three.js for visualization
- Systemd service management
- Blockchain-based audit logging
